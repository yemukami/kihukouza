
## 3. 現在議論中の論点

### 3.1 技組の法人格（株式会社化は「前提条件付き」に修正）

**重要**: handoff.mdの指摘により、株式会社化は「有力候補」から「前提条件が整えば検討」に修正。

**株式会社化の前提条件**:
- [ ] 収益モデルが成立する見込みがある
- [ ] IP/成果物の帰属ルールが合意されている
- [ ] コンソとの権限境界が明文化されている
- [ ] 大学の縛り（利益相反等）がクリアできる
- [ ] 先行企業間で初期持分の合意がある

### 3.2 IP/成果物の帰属（新規追加）

| 成果物の種類 | 帰属先 | 利用許諾 |
|--------------|--------|----------|
| 1系で生まれた知見 | コンソの共有財産 | メンバー自由利用 |
| 2系で生まれた標準 | 技組の管理下 | メンバーに許諾 |
| 個社の既存IP | 各社帰属 | 個別契約 |
| 共同開発の新規IP | 共有 or 技組 | 事前合意 |

### 3.3 運営費モデル（新規追加）

```
Phase 1（立ち上げ期）: 県助成が主、会費は低額
Phase 2（自立期）: 県助成縮小、手数料・案件上乗せを導入
Phase 3（持続期）: 県助成なし、自立
```

### 3.4 報酬設計

**現状の問題**:
- [A] 弁当参加企業: 無報酬
- [B] 成果物納品企業: 報酬あり
- コンソ/技組移行後は大学からの支払いなし

**対策**:
- 短期: 貢献台帳の整備、可視化、優先権の約束
- 中期: 将来収益配分、法人化時の持分

### 3.5 2系→1系の成果物フロー

1系が打上げ後に教育用供給に移行する際、2系の標準モジュールが必要だが、時期的に間に合うか不明。

## 11. 論点（統合）

  ### 11.1 コンソ主体の位置づけ
  - 暫定は大学（寄附講座）主導。将来は任意団体/一般社団法人への移行可否を検討。
  - 大学主導の期間は「学内規程・利益相反・資金管理」を前提に運用する。

  ### 11.2 権限境界とRACIの整合
  - 総会/運営委員会/技術WGの分掌とRACIを一致させる。
  - 重大変更の閾値（コスト/期間/互換性/対外契約）を定義する。

  ### 11.3 技組の法人格（前提条件付き）
  - 収益モデル/IP帰属/権限境界/大学縛り/初期持分合意が成立した場合のみ法人化を検討。
  - 法人格の候補: 当面なし/合同会社/株式会社。

  ### 11.4 IP・成果物の帰属
  - 1系の実績知見: コンソ共有財産。
  - 2系の標準資産: 技組管理下で利用許諾。
  - 共同開発の新規IP: 事前合意で共有 or 技組帰属。
  - 個社既存IPは各社帰属（個別契約）。

  ### 11.5 報酬・先行企業配慮
  - 原資不足のため「貢献台帳＋可視化＋優先権」を最短で実装。
  - 将来は収益配分/持分付与の制度化を検討。

  ### 11.6 1系/2系の成果物流れと時期競合
  - 2系標準が1系の教育供給に間に合わない可能性がある。
  - 対策案: 1系は当面買い物コンポ/2系一部モジュール先行開発/教育供給の後ろ倒し。

  ### 11.7 命名体系
  - 1系/2系の名称（BSP/SSP/CSP等）の混同回避が必要。
