# 地上局計画

## 概要

地上局サブプロジェクトのサクセスクライテリアとWBS。

**本計画のゴール: Stage 3（BSP-01との通信）達成**
**目標期間: 1〜1.5年でBSP-01待ち状態へ**

整理：
- Stage 3 = **県内需要のダウンリンク**（BSP-01/UHFを核に成立させる）
- Stage 4 = **商用運用（ArkEdge連携）**（S/X帯、ネットワーク参加型、別計画）

---

## 段階構成（サクセスクライテリア）

```
【Stage 1】微弱電波取得テスト
├ 目的: RF系全体の基礎検証
├ 方法: 減衰電波での室内テスト、または地上遠距離通信
└ 成功基準: 微弱信号の取得・復調

      ↓

【Stage 2】既存衛星テレメトリダウンリンク
├ 目的: 衛星追尾 + 受信系の検証
├ 対象: ISS、アマチュア衛星等
└ 成功基準: 衛星信号の受信・デコード

      ↓

【Stage 3】BSP-01と通信
├ 目的: 自機との通信成立
├ 対象: BSP-01
└ 成功基準: テレメトリ受信 + コマンド送信

      ↓

【Stage 4】商用運用（ArkEdge連携 / エクストラサクセス）
├ 目的: ArkEdge主導のGSaaSネットワークに参加し収益化
├ 対象: ArkEdge経由の外部顧客（LEOコンステレーション等）
├ 成功基準: ネットワーク稼働・利用料シェア
├ 山形の価値: 日本海側カバレッジ、災害バックアップ、廃校利用
└ ※S/X帯を想定。Stage 3とは設備・免許が異なるため別計画として扱う
```

## Stage 4: 商用運用（ArkEdge連携）

### 基本方針

- **商用展開はArkEdge Space主導**を想定
- 山形は「日本国内マルチサイト地上ネットワーク」の一拠点として参加
- BSP-01（UHF）実績をベースに、S/X帯対応を段階的に拡張
- Stage 3とは設備・免許が異なるため、別計画として扱う

### 市場環境（2025-2026年調査時点）

| 指標 | 数値 |
|------|------|
| GSaaS市場（APAC） | CAGR 21.1%で最速成長 |
| 衛星地上局全体 | 2025年$410億 → 2030年$827億 |
| 日本LEO需要 | 今後10年で80%拡大見込み |

### ArkEdge Spaceの地上局戦略

ArkEdgeは「日本国内マルチサイト地上ネットワーク強化」を明言：

| 拠点 | 状況 |
|------|------|
| 静岡・牧之原 | 稼働中（3.9mアンテナ、S/X/Ka） |
| 北海道・大樹 | 稼働中（VDES海上通信） |
| **山形（候補）** | BSPプロジェクトから提供 |

パートナーシップ:
- SKY Perfect JSAT: 地上局相互利用、GSaaS（2025年2月）
- KSAT: グローバル展開、地上局共同開発（2025年6月）

### 山形が提供する価値

1. **日本海側カバレッジ** - 北海道・静岡の2拠点は太平洋側寄り。本州日本海側の補完。
2. **災害バックアップ** - 南海トラフ・首都直下地震時の冗長性。山形内陸は津波リスクなし。
3. **3拠点体制** - 北海道・山形・静岡で日本列島を面でカバー。
4. **廃校利用** - 設備コスト削減 + 地域連携ストーリー。

### 収益モデル（想定）

```
ArkEdge主導のGSaaSネットワーク
        │
        ├─ 北海道・大樹
        ├─ 静岡・牧之原
        └─ 山形 ← ネットワーク利用料シェア
```

- 単独での商用運営は投資回収が困難
- ネットワーク参加型でリスク分散
- 詳細は ArkEdge との協議で確定

### 前提条件

- [ ] ArkEdge Spaceとの協議・合意
- [ ] 設備仕様の確定（アンテナサイズ、周波数帯）
- [ ] 設置場所の電波環境確認（見晴らし、干渉、冬季稼働率）
- [ ] 免許取得方針（S/X帯）
- [ ] 投資・収益計画の合意

### 技術的制約

| 周波数帯 | 降雨減衰 | 山形での運用 |
