## 2. 想定フォーマット

### 2.1 案A：体験版（45分×複数回）

```
┌─────────────────────────────────────────────────────────┐
│ 【体験版】 45分（1日3-4回転を想定）                       │
├─────────────────────────────────────────────────────────┤
│ 0:00-0:05  導入「山形の衛星、知っていますか？」          │
│                                                         │
│ 0:05-0:15  制約カード説明（4枚）                        │
│            「1年で消える」「届くかわからない」           │
│            「数分しか話せない」「1基1,000万円」          │
│                                                         │
│ 0:15-0:25  ミニCrazy Eights（5分で5案）                │
│            お題：「1年で消える衛星に何を託す？」         │
│                                                         │
│ 0:25-0:35  共有・投票                                  │
│            → 壁に貼り出し、シール投票                   │
│                                                         │
│ 0:35-0:45  まとめ＋宙の種への誘導                       │
│            「興味があれば10月からの本編へ」              │
│            → QRコードでLINE/メール登録                 │
└─────────────────────────────────────────────────────────┘
```

**メリット**：回転率高い、敷居低い
**デメリット**：深い議論にならない

---

### 2.2 案B：本格版（2時間×1回）

- `産業の芽.md`の初回プログラム（30分ハンズオン＋90分本編）をそのまま実施
- 事前予約制、定員15-20名
- 10月以降の参加候補者を獲得

**メリット**：本番と同じ内容で検証できる
**デメリット**：来場者の参加ハードルが高い

---
